What is PDX Code Guild?

PDX Code Guild is an immersive training program that gives students the skills and tools to become a developer. Students build their portfolio and receive individualized attention because of our small class sizes.

PDX Code Guild is creating a network of developers and shaping the future of Oregon’s technology ecosystem.
